What is Loep?

Loep is an open source program for monitoring linux events and resources. It can create a static html page for presentation system logs such as Authentication, ssh, tinyproxy and other services. Please visit live demo using this link this link.

The installation is very simple and easy and did not need meny dependency and library. Before installation please watch tutorial video using this link this link.

Features

  • Easy installation
  • Auto check new version
  • Auto check dependency
  • Configuration ability
  • Network Heartbeat
  • Monitoring Bandwidth Usage
  • Monitoring Servers Resources
  • Monitoring Authentication Logs
  • Monitoring Services logs
