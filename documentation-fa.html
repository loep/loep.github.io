سورس برنامه را از کجا دانلود کنم ؟

سورس برنامه بر روی گیت هاب منتشر شده است. همچنین می توان از لینک های زیر استفاده کرد :

https://github.com/loep/loep/archive/master.zip

چگونه فایل master.zip را از حالت فشرده خارج کنم ؟

ما از دستور unzip در سیستم عامل لینوکس استفاده می کنیم :

$ unzip master.zip

نصب بر روی توزیع debian :

# apt-get install unzip

نصب بر روی توزیع centos :

# yum install unzip

چگونه پیش نیاز ها را بر روی سیستم بررسی کنم ؟

ابتدا پرمیشن اجرایی برای install.sh در نظر می گیریم :

# chmod +x install.sh

سپس با استفاده از سویچ c- پیش نیاز ها را بررسی می کنیم :

# ./install -c

نحوه نصب پیش نیاز ها بر روی debian به چه شکل است ؟

با استفاده از دستور apt-get . به عنوان مثال :

# apt-get install sysstat
# apt-get install bc
# apt-get install curl
...

نحوه نصب پیش نیاز ها بر روی centos به چه شکل است ؟

با استفاده از دستور yum . به عنوان مثال :

# yum install sysstat
# yum install bc
# yum install curl
# yum install expr
...

چگونه برنامه را نصب کنم ؟

با استفاده از سویچ i- :

# ./install -i

چگونه برنامه را آپدیت کنم ؟

مطابق با همین راهنما، ابتدا برنامه را به صورت کامل پاک کنید. سپس ورژن جدید را دانلود و نصب کنید.

چگونه برنامه را پاک کنم ؟

ابتدا دایرکتوری اصلی برنامه را پاک کنید :

# rm -rf /opt/loep_v1.0/

سپس دایرکتوری ای که فایل های html در آن قرار دارد را پاک کنید. به عنوان مثال :

# rm -rf /var/www/html/loep/

در آخر اسکریپت ها را از داخل فایل crontab پاک کنید :

# nano /etc/crontab
