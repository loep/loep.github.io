How to download source code ?

You can download source code from github or other mirror. Please visit download page.

http://github.com/loep/
http://github.com/loep/master.zip
http://loep.ir/loep/master.zip
http://oslearn.ir/loep/master.zip

How to extract master.zip file ?

We use 'unzip' command in linux distributions :

$ unzip master.zip

Install 'unzip' in debian distribution :

# apt-get install unzip

Install 'unzip' in centos distribution :

# yum install unzip

How to check dependencies on system ?

In the first step, set execute permission for install.sh :

# chmod +x install.sh

Then, check dependencies, using -c switch :

# ./install -c

How to install dependencies on debian ?

By using apt-get command; for example :

# apt-get install sysstat
# apt-get install free
# apt-get install users
# apt-get install rev
# apt-get install expr
# apt-get install ifconfig
# apt-get install curl
# apt-get install wget
# apt-get install bc
# apt-get install unzip
# apt-get install uptime

How to install dependencies on centos ?

By using yum command; for example :

# yum install sysstat
# yum install free
# yum install users
# yum install rev
# yum install expr
# yum install ifconfig
# yum install curl
# yum install wget
# yum install bc
# yum install unzip
# yum install uptime

How to install loep ?

By using -i switch :

# ./install.sh -i

How to uninstall loep ?

First step, Remove program's directory :

# rm -rf /opt/loep_v1.0/

Next step, Remove program's html pages from webserver. for example we remove loep directory from webserver :

# rm -rf /var/www/html/loep/

At last, remove scripts from crontab file :

# nano /etc/crontab
